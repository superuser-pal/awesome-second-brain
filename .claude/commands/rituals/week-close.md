description: "Close the active week — calculate velocity, handle incomplete tasks, consolidate daily notes into the week file, archive to plan/archive/."
---

Run the `weekly-closing` workflow from the `daily-rituals` skill.

1. Load the active week file from `plan/`
2. Calculate velocity (completed / planned)
3. Handle each incomplete task: carry forward, backlog, drop, or pause
4. Gather retrospective (what went well, what to change)
5. Consolidate daily notes from `plan/DD-MM-YY-*.md` into the week file
6. Archive week file to `plan/archive/W[x].md`
