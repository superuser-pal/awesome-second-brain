# What If Scenarios

## Strategy

Identify the key assumption or variable in the current situation. Generate 3 alternative "what if" scenarios by varying that assumption (best case, worst case, and a plausible surprise). For each scenario, trace the downstream implications: what changes, what new risks or opportunities emerge, what decisions would look different. Return the scenario map with implications and how the current plan holds up against each.

## When to apply

- Contingency planning where the future is genuinely uncertain.
- Strategy reviews where a key dependency (market, person, resource) could change.
- Note synthesis where multiple interpretations of the same data are plausible.
- Before committing to a plan that has a single point of failure.

## Trade-offs

- **Pro**: Builds strategic flexibility — the user sees the plan from multiple futures.
- **Pro**: Surfaces options that only become visible under changed conditions.
- **Con**: Can create analysis paralysis if too many scenarios are explored.
- **Con**: Scenarios are only as good as the assumptions they vary — blind spots persist.
