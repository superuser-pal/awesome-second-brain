# Competencies

One section per competency from the organization's competency framework. Evidence accumulates via wikilinks from work notes.

## How It Works

1. Add a new `## [Competency Name]` section when a competency is first encountered
2. Link work notes to competencies in their `## Related` section: `[[work/05_REVIEW/COMPETENCIES#Competency Name]]`
3. Evidence accumulates via backlinks — check backlinks panel for all work that demonstrates a competency
4. Use `current-level` and `target-level` notes per section to track progression
5. Review prep: scan this file when preparing self-assessments
