# Home Charging Milestone: UI Flow Shipped

Shipped the first working version of the home charging onboarding flow to the pre-production Toyota EV test fleet. The flow covers wallbox pairing, tariff setup, and schedule configuration. No critical bugs in first 48h. Squad celebrated at end of sprint.

## Key Decisions

- Simplified the tariff input screen after user testing showed confusion with dual-rate input
- Deferred smart charging scheduling to next sprint based on stakeholder input
- Agreed with [[work/06_ORG/PEOPLE#Damiano Valente|Damiano Valente]] that the sprint scope was locked 1 week early to avoid last-minute scope creep
